Déploiement via Helm (La méthode standard)
Pour installer l'opérateur Percona (le plus robuste pour MySQL HA) :

Installer l'opérateur :

Bash
helm repo add percona https://percona.github.io/percona-helm-charts/
helm install my-db-operator percona/percona-xtradb-cluster-operator

Créer ton Cluster HA (Custom Resource):

kubectl apply -f mysql-ha.yaml


Partie B: Redis en mode Sentinel (HA)

kubectl create secret generic redis-password \
  --from-file=password=./redis-secrets.txt

helm repo add redis-operator https://spotahome.github.io/redis-operator
helm install redis-operator redis-operator/redis-operator

kubectl apply -f redis-ha.yaml


Partie C: Redis en mode Sentinel (HA)

helm repo add minio https://charts.min.io/
helm install minio-ha minio/minio \
  --set mode=distributed \
  --set replicas=4 \
  --set rootUser=admin \
  --set rootPassword=MotDePasseAdminMinio \
  --set persistence.size=10Gi
